JAVA의 리스트
    1. 선언
        ArrayList<타입> 리스트변수명 = new ArrayList<>();
        ex) 문자열 리스트 : ArrayList<String> 리스트변수명 = new ArrayList<>();
        ex) 정수 리스트 : ArrayList<Integer> 리스트변수명 = new ArrayList<>();
        ex) 실수 리스트 : ArrayList<Double> 리스트변수명 = new ArrayList<>();
            int -> Integer : 기본 자료형을 클래스처럼 사용할 때 사용되는 래퍼클래스
                - 기본 자료형은 리터럴만 다루기 때문에 추가적인 기능/속성을 다루기 위해 클래스화
        ex) 사용자 정의 타입 리스트 : ArrayList<Member> 리스트변수명 = new ArrayList<>();
            - 사용자 정의 타입 : 내가 만든 클래스
    2. 리스트내 요소 추가
        리스트변수명.add(객체);         : 해당 객체를 리스트내 추가
    3. 리스트내 요소 호출
        리스트변수명.get(인덱스);        : 해당 인덱스의 요소(객체)반환
    4. 리스트내 요소 삭제
        리스트변수명.remove(인덱스)      : 해당 인덱스의 요소(객체) 삭제, 삭제된 요소 뒤로 한칸씩 당겨짐.
        리스트변수명.clear();           : 리스트내 전체 요소 삭제
    5. 리스트내 요소 총개수
        리스트변수명.size()             : 리스트내 요소 총개수
    6. 리스트내 요소 찾기
        리스트변수명.indexOf(값 또는 객체)     : 리스트내 값 또는 객체가 존재하면 찾은 인덱스 반환 없으면 -1
        리스트변수명.contains( )              : 리스트내 값 또는 객체가 존재하면 true 없으면 false
    7. 리스트내 비어있는지 확인
        리스트변수명.isEmpty( )               : 리스트내 요소가 하나도 없으면 true 1개라도 존재하면 false
    8. 리스트 순회(리스트내 첫번째 요소/인덱스부터 마지막 요소/인덱스까지 호출)
        1.
            for( int i = 0 ; i < 리스트변수명.size() ; i++ ){
                System.out.println(리스트변수명.get(i););
            }
        2.
            for( 타입 반복변수명 : 리스트 ) {System.out.println(반복변수명);}

        3.
            리스트변수명.forEach( 반복변수명 -> System.out.println(반복변수명) );
