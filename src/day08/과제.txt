- 타입이란?
변수 , 메소드 등에서 사용되는 데이터의 종류를 뜻합니다.

- 타입을 언제 사용하는지?
변수를 선언하거나 데이터를 사용할 때 낭비되는 메모리를 줄이기 위해서 타입을 구분지어 둡니다.

- 클래스란?
객체의 속성/행위들을 묶어둔 틀입니다.

- 클래스를 언제 사용하는지? 
객체를 생성하기 위해 사용하며 클래스라는 하나의 틀로 묶어두기 때문에 코드 가독성과, 유지보수성이 좋습니다.

- 객체란?
우리가 보고 인지하는 모든 것을 뜻합니다.

- 객체를 언제 사용하는지?
여러 속성의 데이터를 효울적으로 관리하기 위해서 객체를 생성합니다.

- 멤버변수란?
클래스 내부에 선언되며 객체의 특징을 나타내는 속성을 말합니다.

- 멤버변수를 언제 사용하는지?
객체를 생성할 때 객체의 속성을 정할  때 사용합니다.

- 생성자란?
객체를 생성할때 호출되는 메소드입니다.

- 생성자를 언제 사용하는지?
클래스를 처음 만들 때 멤버 변수들을 초기화 하기 위해서입니다.

- 메소드/함수란?
여러 실행코드들을 묶어두는 틀입니다.

- 메소드/함수를 언제 사용하는지?
- 반복되는 코드들을 줄이기 위해 사용합니다.

- 매개변수란
함수내에서 사용되는 변수로 함수를 호출하는 곳에서 전달하는 값입니다.

- 매개 변수를 언제 사용하는지?
함수를 호출할 때 함수내로 데이터를 전달하기 위해서 

- 반환타입이란?
리턴할 값의 타입입니다.

- 반환/리턴이란?
함수내 결과를 호출한 곳으로 전달할 값

- 반환/리턴을 언제 사용하는지?
함수가 종료될 때 사용 되며 함수내에서 호출할 곳으로 결과값을 전달하기 위해 사용합니다.
